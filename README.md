# FutureDrive

Quote-based ride booking (Autos & Cabs) — MVP scaffold for Neemrana/Behror corridor.

This repository contains the initial project scaffold created from the product SRS and an MVP-first prioritized backlog. Use this repo as the starting point for development, issue creation, and further design artifacts.

What’s included in this commit:
- BACKLOG.md — Prioritized backlog (Epics → Stories → Acceptance Criteria)
- docs/SRS.md — Updated Software Requirement Specification (from product owner)
- api/openapi.yaml — placeholder OpenAPI file for core auth endpoints
- db/schema.sql — initial SQL schema for core entities
- i18n/en.json and i18n/hi.json — i18n skeleton (English + Hindi)
- .gitignore — Node / Android / iOS ignores
- LICENSE — MIT

Getting started
1. Clone the repo: git clone https://github.com/yadavrahulcse/futuredrive.git
2. Review BACKLOG.md for prioritized stories to convert into issues.
3. Expand api/openapi.yaml and db/schema.sql as backend dev work proceeds.
